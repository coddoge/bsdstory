# 个人评价

在记忆溯行-学园篇中，甲在学园时的经历按时间顺序展现在玩家面前。

学园篇中隐藏了很多伏笔，并将这些伏笔中的一部分巧妙地安排在了各个线路中。
部分玩家对在主线中穿插的记忆溯行并不感冒，部分是因为很多青春气息浓厚的部分放到子弹纷飞的现代篇中去回忆显得格格不入。
尽管如此，也正是因为有着这样美好而青涩的回忆，才更加衬托出现代篇中众人的坚持和痛苦。
对其的主观感受也许更多是篇幅上的接受度的原因。
此外，在作为攻略过程中阅读学园篇的时候，相当部分的故事是已经在主线中阅读的，反而导致心急的玩家容易忽视掉部分夹杂着的新内容。而且，在Dive 1+2联动时，玩家会在蕾线便早早开启记忆溯行(尽管仍然会按篇章逐渐开启)，误入的玩家容易因此影响体验。

学园篇是开启真线的必要条件，其理由是在真线中，甲并未如同之前玩家经历的四条线路一样进入失忆状态。
而在学园篇的故事中，半数情节在之前的四条线路中已被揭示。
反过来说，也提供了相当多玩家并没有机会在之前的线路中了解的记忆。
这部分记忆中，少数是细节上的补充，多数则是与真和空相关的记忆，被隐藏有着节奏、关联性和剧透等各方面的原因。

在学园篇中，玩家得以更立体的认知甲和空等主要角色的形象。
在主线中令人迷惑的呼的封印、甲与空的交往经历也在此处揭晓。
温柔而不自觉的甲为了保持如月寮的平衡微妙地把握着自己与众人的距离。
热心而又敏锐的空表面上总和甲吵架不断，但总在细微之处表现出谢意和关怀。
由于空的最终胜出多少有共振现象的加成，因此若单看学园篇的描写，甲和空的恋情还略显不服(by 还没认输的千夏)。
但行文间也能感受到，共振现象更多是用来打开略显别扭的两人的心结。
假如甲和空换一种方式相遇的话，也许两人一开始便能互通心意了吧。

在学园篇中，水无月真的形象也逐渐饱满起来。
对比之前的主线，真很少出现在被其视为哥哥的甲的回忆中。
不仅如此，她也是唯一一个至今仍在现代篇中下落不明的人(尽管从立绘和语音上基本可以立刻推断出来身份)。
现实中寡言少语、网络中却如同机关炮一般的真感情实际上十分细腻，但性格却很执拗，并且容易将过错归因到自己。
甲将如何拯救这样的她，将会在接下来开启的真线中揭晓。
